Лабораторная работа 4 - Сбалансированные деревья
Реализовано красно-черное дерево (вариант 3, так как мой номер 2: (2%3)+1=3)

Как собрать

make clean
make

Как запустить

Интерактивный режим:
./lab4

Из файла в файл:
./lab4 input.txt output.txt

Доступные команды

1 <ключ> <значение> - добавить узел
2 <ключ> - удалить узел
3 - напечатать дерево
4 <ключ> - найти значение по ключу
help - показать справку
exit или quit - выйти

Пример работы

./lab4
> 1 abc 3.14
Inserted: abc = 3.14
> 1 xyz 5.5
Inserted: xyz = 5.50
> 3
Tree (R=red, B=black):
    xyz[R] (5.50)
abc[B] (3.14)
> 4 abc
Found: abc = 3.14
> exit
Goodbye!
